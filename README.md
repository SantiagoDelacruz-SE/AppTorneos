# AppTorneos

Aplicación desarrollada en Python para la gestión integral de torneos y competiciones deportivas.

## 🎯 Características

- Gestión completa de torneos deportivos
- Interfaz con animaciones integradas
- Múltiples módulos de aplicación para diferentes funcionalidades
- Sistema visual atractivo con soporte para GIFs animados

## 🚀 Tecnologías Utilizadas

- **Python** - Lenguaje principal de desarrollo
- Animaciones personalizadas

## 📁 Estructura del Proyecto

```
AppTorneos/
├── app.py                      # Aplicación principal
├── app2.py                     # Módulo secundario
├── apptorneos.py              # Módulo principal de torneos
├── Apptorneosanimacion.py     # Módulo de animaciones
├── ganador.gif                # Animación de victoria
└── .idea/                     # Configuración del IDE
```

## 💻 Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/SantiagoDelacruz-SE/AppTorneos.git
```

2. Accede al directorio:
```bash
cd AppTorneos
```

3. Ejecuta la aplicación:
```bash
python app.py
```

## 📝 Uso

La aplicación permite gestionar torneos deportivos, mostrando animaciones visuales para los ganadores y manteniendo un control de las competiciones.
